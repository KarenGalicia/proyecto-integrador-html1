## Descripción
Estructura HTML5 semántica de la página principal de un blog personal, 
aplicando buenas prácticas de accesibilidad (jerarquía de encabezados, 
navegación por teclado) y organización correcta del documento mediante 
etiquetas semánticas (header, nav, main, section, article, aside, footer).

## Validación W3C

(https://validator.w3.org/) y no presenta errores ni warnings.
